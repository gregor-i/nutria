Introduction to Nutria

Nutria contains of 3 Components: The Explorer, the Gallery and the Template Editor.

The Explorer is an iterative Tool to explore a fractal and to try out new configurations. Most fractals are configurable. For example are most fractals bound by a maximal number of iterations. Changing this value is easy and the result is quickly visible. Additionally functions can be used as parameters. For example the famous Mandelbrot iterations is z*z + lambda, changing this function to z*z + 0.285 + 0.01 * i yields an julia fractal.

Explored fractals can be downloaded or saved. Saved fractals will be listed in My Gallery. Exploration can be conveniently continued from there. Fractals can also be published from there. Published Fractals will be listed in the Public Gallery and all users might see them and continue to explore.

The third component is the Template Editor. It is intended to be used by experienced users as it required some insight into Nutria and into programming. All fractals in Nutria are based on templates. Templates define the executed code and all possible parameters.

Privacy Statement

All information Nutria saved about logged in users are visible on My Profile. The Delete Profile Button will delete all information. Nutria does not save any information about not logged in users.
